**Informacje:**
W projekcie mamy symulację biblioteki. Developer w łatwy sposób może stworzyć tablicę z jego wirtualną biblioteką (kolekcją książek). Jest łatwy dostęp do informacji o książce (dostępność, autor, liczba stron itp.). Klasy są odpowiednio ,,oplombowane" przez pola, które są prywatne. Klasy nie posiadają żadnych ,,setterów", ale dostępne są ,,gettery".
```getTytul() oraz isDostepne()```
